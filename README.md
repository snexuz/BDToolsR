## R function
- API URL： http://taibif.org.tw/BDTools/
- 使用說明：transl(sourceSystem, targetSystem, x, y)
- 參數說明：
  - sourceSystem: 來源坐標系統
  - targetSystem: 欲轉換的座標系統
    - 1: WGS84 經緯度
    - 2: TWD67 經緯度
    - 3: TWD97 經緯度
    - 4: TWD97/ 澎湖地區
    - 5: TWD97/ 臺灣地區
    - 6: TWD67/ 澎湖地區
    - 7: TWD67/ 臺灣地區
  - x: X 坐標 lon 東西經度, longitude E
  - y: Y 坐標 lat 南北緯度, latitude N
- 回傳：轉換後的經緯度 dataframe

transl(5, 1, 123456, 2234567)
